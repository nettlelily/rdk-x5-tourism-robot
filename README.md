项目名：基于RDK X5的双足交互机器人。
包含3个子目录：基础双足运动_Demo、Claude智能问答_Demo、指令拍照与上传_Demo。

基础双足运动_Demo
是机器人运动控制的运行代码。机器人采用双足结构设计，具备基础的物理空间移动能力，能够稳定执行直行、左转、右转动作指令。详见对应子目录内的说明文档。

Claude智能问答_Demo
是机器人智能交互的运行代码。系统后台接入了 Claude 大语言模型 API。机器人能够作为交互中枢，实时调用 Claude 分析并回答游客提出的各类问题，提供流畅的咨询体验。详见对应子目录内的说明文档。

指令拍照与上传_Demo
是机器人视觉采集与网络传输的运行代码。设备配备视觉采集模块并支持指令触发。当接收到“拍照”输入指令时，机器人会调用摄像头完成拍摄，随后将生成的图片自动上传至预先配置好的指定网络服务端地址。详见对应子目录内的说明文档。
